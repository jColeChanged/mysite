Extendable

The complicated and configurable Camfrog bot

Extendable lets you make as many question and answer style games as you want! You can set it up to unscramble words, give math questions, or do trivia. The limit is your imagination. And best of all its free! I can't promise you that it will always work. I can't promise you that it will be easy to set up. I can tell you that it has been used and loved in the wild.

Installing The Extendable Module

Step 1: Download the module.
Step 2: Open up Camfrog's CamFrog Bot program and login to your bot.
Step 3: Click add module and select the module you downloaded.

Configuring The Extendable Module

Step 1: Go to your Documents folder. Or any folder really.
Step 2: Create a folder in the folder you went to named bot_plugins.
Step 3: Go inside the folder you just created.
Step 4: Create a file named directory_listing.txt.
Step 5: Right click the file you made and select properties.
Step 6: Highlight the path to the file and copy it.
Step 7: Open your CamFrog bot and go to the Extendable module's settings page.
Step 8: Paste the path you copied earlier into the path box.
Step 9: Click Set. Click apply. Activate and deactivate the module.

If you did this right, nothing exciting will happen. You can now add games to your bot. Congratulations.

Adding A Game

There are two ways to add a game: make the game yourself, or use one of the ones I've made for you. If you want to try to make your own game, read this. Otherwise:

Step 1: Download the game you want to install.
Step 2: Unzip the downloaded folder into the bot_plugins that you created when installing the program.
Step 3: Open the folder you just unzipped. Right click the questions.txt file and select properties.
Step 4: Copy the path to the file.
Step 5: Open directory_listing.txt, the file you created when configuring the module. On a new line, paste the path to the file that you just copied. Save and close the file.
